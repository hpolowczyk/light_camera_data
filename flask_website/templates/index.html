Lights! Camera! Data! is a data visualization project based on 250 of the highest grossing movies of all time according to the amount they grossed over their opening weekend.


It explores the relationships between films, actors and studios using a chord diagram. It investigates foreign gross box office success via map.


Word cloud...


Tree Map...


Data was sourced from Box Office Mojo and the OMDb API. It was then transformed using Python and loaded into MongoDB. The visualization is powered by Python Flask.


Visuals were made using d3.js, dex.js, leaflet.js, AnyChart.js and Google Chart.


Created by Amanda Green, Hanna Polowczyk, Manoel Burgos, and Sofiat Ishola.
